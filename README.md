# PageSpeed-CLI [![codecov](https://codecov.io/gh/awcodify/pagespeed-cli/branch/master/graph/badge.svg)](https://codecov.io/gh/awcodify/pagespeed-cli)

CLI for testing a web performance using Google PageSpeed

## Installation

```bash
go build 
```

## Usage

```bash
./pagespeed-cli --help
```
